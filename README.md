# SimpleHTTPServer
Невеличкий HTTP сервер який було створено у навчальних цілях.

# Збірка

**Linux**
``` 
$make
```
**Windows**

Необхідно встановити **MinGW** (рекомендую використовувати дистрибутив [w64devkit](https://github.com/skeeto/w64devkit))

```
$make win
```
